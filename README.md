# Елементи UI
## Text
```
    {
        "type":"text",
        "x": int,
        "y": int,
        "w": int,
        "h": int,
        "text":string,
        "bg":string
    }
```

## Button
```
    {
        "type":"button",
        "x": int,
        "y": int,
        "w": int,
        "h": int,
        "text": string,
        "cmd": string
    }
```

## Slider
```
    {
        "type":"slider",
        "x": int,
        "y": int,
        "w": int,
        "h": int,
        "color": string,
        "cmd": string,
        "autoconter": bool,
    }
```

## Сценрій
```
    {
        "cmd": string,
        "type": "tougle/click/none",
        "actions": 
            [
                {
                    "type": 
                    "cmd": string,
                },
                {
                    "type": "motor",
                    "a": string (source of A half bridge values),
                    "b": string (source of B half bridge values),
                    "weight": int (in grams)
                },
                {
                    "type": "blink",
                    "points": [
                        {"pin": "1", "offset":"0", "value":"0"},
                        {"pin": "1", "offset":"500", "value":"255"},
                        {"pin": "1", "offset":"1000", "value":"0"}
                    ]
                }
            ...
            ]

    }
```
